Requests for access

For copyright and personal rights reasons, access to the holdings of the Thomas Mann Archive is in many cases subject to authorisation. Please note the access conditions that are specified for each document.

To submit a request for access to a protected document, you must first register and log in. Please place the desired dossier or individual document in the order basket and then select the option “Submit request for access”. The archive will then assess your application. If necessary, additional access authorisations will be requested from the respective rights holders.

You can usually expect a decision within two weeks. The processing time may be longer for complex applications.

Please note that permission to view does not constitute permission to publish. Citations must be authorised separately.
